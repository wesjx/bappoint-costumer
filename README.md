# Bappoint costumer page
This is a simple way to get you started quickly.

---

## Prerequisites

- Node.js 18.18 or higher  
- npm or pnpm package manager  
- Internet connection to download dependencies

---

## Installation

Clone the repository and install dependencies:

git clone https://github.com/wesjx/bappoint-costumer.git

cd your-project

pnpm install

---


---

## Running in Development Mode

Start the development server to run the app locally:

pnpm run dev


---

## Project Structure

- `/app` - main routes and components (if using App Router)  
- `/components` - reusable UI components  
- `/public` - static files like mocks, images and icons  
- `/styles` - CSS or styling files
